Título e parágrafo

A tag (h1) é usado para títulos e (h2) para subtítulos e temos os perspectivos (h3 e h4...) para textos mais complexos.

Podemos usar a tag (p) que signifíca parágrafo e dentro desta tag podemos escrever oque quisermos,
também podemos quebrar um parágrafo onde quisermos, como este por exemplo basta usar tag <br>

favicon

"favicon"é o emblema que fica no topo da aba do navegador, para usalo usamos a Tag sobre o TITLE no HTML5, vemos um exemplo a abaixo. 

        
                    link rel="shortcut icon" href="sua favicon" type="favicon"
                    <title> Seu título  </title>
                    </head>

Para fazermos este exemplo acima usamos a tag (pre) e dentro dessa tag usamos a tag (code) desse modo podemos simular o código em um parágrafo.

Listas

Podemos usar varias listas, temos listas ordenadas, listas não ordenadas e lista de definição

lista ordenada

para criar uma lista ordenada usa-se a tag <ol> e dentro dessa tag usa-se a tag <li>

essa lista pode ser usada com os type="1 / A / a / I / i"

  1. Acordar
  2. escovar os dentes
  3. Estudar
  4. Almoçar

Lista não ordenada

para criar uma lista não ordenada usa-se a tag <ul> e dentro dessa tag usa-se a tag <li>

essa lista pode ser usada com os type="square / circle / disc "

  • pao
  • Queijo
  • café
  • leite

Lista de definição

para criar uma lista não ordenada usa-se a tag <dl> e dentro dessa tag usa-se as tags <dt> e <dd>

a tag <dl> signifíca lista de definição, a tag <dt> signifíca termo e a tag <dd> signifíca descrição.
